Detailed Description

UART driver implementation for handling serial transmission in the shell.

Provides UART receive and transmit functions, supporting both simple (blocking) and interrupt-based (non-blocking) modes. This file also manages task notifications for completion in interrupt mode, using FreeRTOS task handles.
