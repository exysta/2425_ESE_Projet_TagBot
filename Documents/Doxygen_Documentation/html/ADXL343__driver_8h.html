Detailed Description

Driver for ADXL343 accelerometer.

This header file contains the register definitions, function prototypes, and task management for the ADXL343 accelerometer. The functions allow for the initialization, configuration, reading, and writing to the ADXL343 registers. Additionally, functions for detecting taps and reading accelerometer data (X, Y, Z axes) are provided.
